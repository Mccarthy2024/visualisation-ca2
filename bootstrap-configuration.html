Bootstrap Configuration – Portfolio Website

This page outlines the Bootstrap features and components used in my one-page portfolio website for Data Visualisation CA2.

1. Bootstrap Features Used

  • Grid system: container, row, col-md-*, col-lg-* for responsive layout in all sections (hero, education, projects, experience, skills, hobbies, contact).
  • Navbar: Responsive navbar with brand, collapse menu and links to each section of the page.
  • Cards: .card components used to present Education details, Projects and Work Experience, with consistent headings and body text.
  • Buttons: Primary and outline buttons (.btn, .btn-primary, .btn-outline-secondary) used in the hero section and for “More details” links on project cards.
  • Collapse: data-bs-toggle="collapse" used on project cards to show and hide extra project details inside a collapsible area.
  • Modal: Bootstrap .modal used for the Contact form dialog, opened with a button in the Contact section.
  • Forms: .form-control, .form-label, .btn, and validation hints inside the contact form.
  • Alerts: .alert and .alert-success used to show a success message after the (fake) contact form submission.
  • Progress bars: .progress and .progress-bar to visualise skills levels in the Skills section.
  • Utility classes: Spacing and text utilities such as .mb-3, .mb-4, .mt-2, .text-center, .text-muted, .bg-light, .shadow-sm, .d-flex, .justify-content-center, etc.
  • Images: .img-fluid and .rounded-circle used for the profile image in the hero section.
